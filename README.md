MediaCenter
===========
 
Usage
-----

Require FFMPEG and FFPROGE

Install FFMPEG
````
sudo apt-get install libavcodec53 libavdevice53 libavformat53 libavutil51 libpostproc52 libswscale2 ffmpeg
````
